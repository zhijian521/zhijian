# 统计接口

- 文档状态：当前实现基线
- 适用范围：当前项目
- 最后核对：2026-07-21
- 维护范围：观澜站点、概览、访问记录和公开采集接口
- 关联代码：`src/app/api/admin/analytics/`、`src/app/api/collect/`

本文档记录当前 5 个接口。后台统计接口需要管理员登录，采集与预检接口公开。

## /api/admin/analytics/overview · 统计概览

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/analytics/overview/route.ts`

### GET · 获取综合统计数据

- Query：`siteId: string` 必填。
- Query：`range?: 7d | 30d | 90d`，默认 `7d`。
- 成功数据：`{ overview, trend, pages, sources, devices, languages, countries, regions, browsers, os, entryPages, exitPages }`。
- 查询前会按需执行日聚合。

## /api/admin/analytics/data · 统计数据清理

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/analytics/data/route.ts`

### DELETE · 清空指定站点统计数据

- Query：`siteId: string`，必填。
- 成功数据：`{ events: number, daily: number }`，分别表示删除的原始事件和日聚合记录数。

## /api/admin/analytics/sites · 统计站点管理

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/analytics/sites/route.ts`

### GET · 获取站点列表

- 请求：无。
- 成功数据：`{ data: TrackSite[], total: number }`。

### POST · 创建站点

- JSON Body：`name: string`、`domain: string`，均必填。
- 成功状态：`201`。
- 成功数据：`{ site: TrackSite }`。

### PUT · 更新站点

- JSON Body：`id: string` 必填；`name?: string`、`domain?: string`、`status?: active | paused`。
- 成功数据：`{ site: TrackSite | null }`。

### DELETE · 软删除站点

- Query：`id: string`，必填。
- 成功数据：`null`。

## /api/admin/analytics/visits · 访问明细

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/analytics/visits/route.ts`

### GET · 分页查询访问记录

- Query：`siteId: string` 必填。
- Query：`range?: 7d | 30d | 90d`、`page?: number`、`pageSize?: number`。
- 默认值：`range=7d`、`page=1`、`pageSize=20`；`pageSize` 最大为 `100`。
- 成功数据：`{ data: VisitRecord[], total: number }`。

## /api/collect · 统计数据采集

- 鉴权：公开
- 路由文件：`src/app/api/collect/route.ts`

### POST · 批量接收埋点事件

- 请求体：兼容 `text/plain` 和 `application/json`，内容必须是有效 JSON。
- Body：`siteId: string` 必填，最长 32 字符；`visitorId?: string`、`sessionId?: string`。
- Body：`events: RawEvent[]` 必填，每批 1～20 条。
- `RawEvent`：`type`、`url`、`ts` 必填；可选 `referrer`、`title`、`duration`、`screen`、`lang`、`ua`、`isNew`、`isSessionStart`。
- 事件类型：仅接受 `pageview | heartbeat | leave`；无效事件会在批次中过滤。
- 限流：令牌桶，每 `siteId` 每秒最多 10 次请求，超限返回 `429` 空响应；实现复用 `src/lib/core/rate-limit.ts`（进程内内存，惰性清理过期 bucket）。
- 成功响应：HTTP `202`，无响应体。
- 其他响应：限流返回 `429` 空响应；格式、站点或服务错误返回带 CORS 头的统一错误 JSON。

### OPTIONS · CORS 预检

- 请求：可携带 `Origin`。
- 成功响应：HTTP `204`，无响应体。
- 响应头：允许 `POST, OPTIONS` 和 `Content-Type`，`Access-Control-Max-Age=86400`。
