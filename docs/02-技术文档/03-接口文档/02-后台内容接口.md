# 后台内容接口

- 文档状态：当前实现基线
- 适用范围：当前项目
- 最后核对：2026-07-18
- 维护范围：后台文章、分类、标签、图片、用户和 SEO 管理接口
- 关联代码：`src/app/api/admin/`

本文档记录当前 14 个接口，全部需要管理员登录。除 ZIP 下载外，“成功数据”指统一响应结构中的 `data` 字段。

## /api/admin/posts · 文章管理（列表/创建）

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/posts/route.ts`

### GET · 分页获取文章

- Query：`keyword?: string`、`status?: all | draft | published`、`page?: number`、`pageSize?: number`。
- 默认值：`status=all`、`page=1`、`pageSize=10`；`pageSize` 最大为 `100`。
- 成功数据：`{ data: AdminPostListItem[], total: number }`。
- 列表字段：`id`、`slug`、`title`、`status`、`categoryName`、`tagNames`、`publishedAt`、`updatedAt`，不返回正文。

### POST · 创建草稿

- JSON Body：`title?: string`，空值时使用“无标题草稿”。
- 成功数据：完整 `Post`；新文章使用 `draft-{timestamp}` Slug。

## /api/admin/posts/[id] · 文章编辑/删除

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/posts/[id]/route.ts`

### PATCH · 更新文章字段

- Path：`id` 为正整数。
- JSON Body：可选字段 `slug`、`title`、`summary`、`content`、`status`、`publishedAt`、`coverImage`、`altText`、`categoryId`、`tags`。
- `status` 仅允许 `draft | published`；`tags` 为标签 ID 数组。
- 成功数据：更新后的完整 `Post`。

### DELETE · 删除文章

- Path：`id` 为正整数。
- 成功数据：`null`。
- 文章不存在时返回 `404`。

## /api/admin/posts/export · 文章导出

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/posts/export/route.ts`

### GET · 导出文章和引用图片

- Query：`id?: number`；不传时导出全部文章。
- 成功响应：`application/zip` 文件流，文件名为 `zhijian-export-YYYY-MM-DD.zip`。
- ZIP 内容：文章 Markdown、引用图片和 `manifest.json`。

## /api/admin/categories · 分类管理

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/categories/route.ts`

### GET · 获取分类列表

- 请求：无。
- 成功数据：`{ data: Category[], total: number }`。

### POST · 创建分类

- JSON Body：`name: string`、`slug: string` 必填，`sort_order?: number` 默认 `0`。
- 成功状态：`201`。
- 成功数据：`{ category: Category }`。

## /api/admin/categories/[id] · 分类编辑/删除

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/categories/[id]/route.ts`

### PUT · 更新分类

- Path：`id` 为数字。
- JSON Body：`name?: string`、`slug?: string`、`sort_order?: number`。
- 成功数据：`{ category: Category | null }`。

### DELETE · 删除分类

- Path：`id` 为数字。
- 成功数据：`null`。

## /api/admin/upload · 图片上传

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/upload/route.ts`

### POST · 上传单张图片

- Content-Type：`multipart/form-data`。
- FormData：`file: File`，必填。
- 约束：仅接受当前上传模块允许的图片类型，最大 `5MB`。
- 成功状态：`201`。
- 成功数据：`Upload`。

## /api/admin/uploads · 图片列表

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/uploads/route.ts`

### GET · 分页获取图片

- Query：`page?: number`、`pageSize?: number`。
- 默认值：`page=1`、`pageSize=20`；`pageSize` 最大为 `100`。
- 成功数据：`{ data: Upload[], total: number }`。

## /api/admin/uploads/[id] · 图片编辑/删除

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/uploads/[id]/route.ts`

### PATCH · 修改图片信息

- Path：`id` 为正整数。
- JSON Body：`original?: string`、`alt?: string`。
- 成功数据：更新后的 `Upload`。

### DELETE · 删除图片

- Path：`id` 为正整数。
- 成功数据：`null`；同时删除数据库记录和对应文件。

## /api/admin/uploads/sync · 图片同步清单

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/uploads/sync/route.ts`

### GET · 扫描服务器上传目录

- 请求：无。
- 成功数据：`{ files: { path: string, size: number }[], total: number }`。

## /api/admin/users · 用户管理

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/users/route.ts`

### GET · 分页搜索用户

- Query：`page?: number`、`pageSize?: number`、`search?: string`。
- 默认值：`page=1`、`pageSize=20`；`pageSize` 最大为 `100`。
- 成功数据：`{ data: User[], total: number }`。

### POST · 创建用户

- JSON Body：`username: string`、`email: string`、`password: string` 必填，`role?: admin | user` 默认 `user`。
- 成功状态：`201`。
- 成功数据：`{ user: { id, username, email, role, status, created_at } }`。

## /api/admin/users/[id] · 用户编辑/删除

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/users/[id]/route.ts`

### GET · 获取用户详情

- Path：`id` 为数字。
- 成功数据：`{ user: User }`。

### PUT · 更新用户

- Path：`id` 为数字。
- JSON Body：`username?: string`、`email?: string`、`password?: string`、`role?: admin | user`、`status?: active | disabled`。
- 成功数据：`{ user: User | null }`。

### DELETE · 删除用户

- Path：`id` 为数字。
- 成功数据：`null`。
- 约束：管理员不能删除当前自己的账号。

## /api/admin/seo/submit · SEO URL 推送

- 鉴权：管理员登录
- 路由文件：`src/app/api/admin/seo/submit/route.ts`

### POST · 推送当前公开 URL

- 请求：无。
- 推送范围：首页、文章列表和全部已发布文章详情 URL。
- 成功数据：`{ totalUrls: number, indexNow, baidu }`，后两个字段为对应搜索引擎的提交结果。
