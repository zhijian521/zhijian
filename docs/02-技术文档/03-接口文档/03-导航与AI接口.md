# 导航与 AI 接口

- 文档状态：当前实现基线
- 适用范围：当前项目
- 最后核对：2026-07-18
- 维护范围：导航数据、用户同步和 AI 对话接口
- 关联代码：`src/app/api/nav/`、`src/app/api/ai/`

本文档记录当前 8 个接口，全部需要有效用户会话。“成功数据”指统一响应结构中的 `data` 字段，SSE 接口除外。

## /api/nav/data · 导航站全量数据

- 鉴权：用户登录
- 路由文件：`src/app/api/nav/data/route.ts`

### GET · 获取当前用户导航数据

- 请求：无。
- 成功数据：`{ bookmarks: Bookmark[] | null, todos: TodoItem[] | null, notes: NoteItem[] | null }`。
- AI 对话历史不包含在此响应中，需单独调用 `/api/nav/chat`。

## /api/nav/sync · 导航站数据同步

- 鉴权：用户登录
- 路由文件：`src/app/api/nav/sync/route.ts`

### POST · 首次登录时同步本地数据

- JSON Body：`bookmarks?`、`todos?`、`notes?`、`chat?`，传入项必须为数组。
- 单项限制：序列化后最大 `1MB`。
- 成功数据：`null`。
- 冲突：数据库中已经存在任一导航数据时返回 `409`，以数据库为准。

## /api/nav/bookmarks · 书签保存

- 鉴权：用户登录
- 路由文件：`src/app/api/nav/bookmarks/route.ts`

### PUT · 覆盖保存书签

- JSON Body：`{ data: Bookmark[] }`，`data` 必填。
- 成功数据：`null`。

## /api/nav/todos · 待办保存

- 鉴权：用户登录
- 路由文件：`src/app/api/nav/todos/route.ts`

### PUT · 覆盖保存待办

- JSON Body：`{ data: TodoItem[] }`，`data` 必填。
- 成功数据：`null`。

## /api/nav/notes · 笔记保存

- 鉴权：用户登录
- 路由文件：`src/app/api/nav/notes/route.ts`

### PUT · 覆盖保存笔记

- JSON Body：`{ data: NoteItem[] }`，`data` 必填。
- 成功数据：`null`。

## /api/nav/chat · AI 对话历史

- 鉴权：用户登录
- 路由文件：`src/app/api/nav/chat/route.ts`

### GET · 获取全部会话

- 请求：无。
- 成功数据：`ChatConversation[]`，没有记录时返回空数组。

### PUT · 覆盖保存会话

- JSON Body：`{ data: ChatConversation[] }`，`data` 必须为数组。
- 数据限制：序列化后最大 `1MB`。
- 成功数据：`null`。

## /api/ai/chat · AI 对话

- 鉴权：用户登录
- 路由文件：`src/app/api/ai/chat/route.ts`

### POST · 发起流式 AI 对话

- JSON Body：`messages` 必填，最多 50 条；每条为 `{ role: user | assistant | system, content: string }`，内容最长 8000 字符。
- JSON Body：`model?: string`，可选，最长 64 字符；不传时使用环境变量或 `deepseek-chat`。
- 成功响应：`text/event-stream`，增量格式为 `data: {"content":"..."}`，结束事件为 `data: [DONE]`。
- 说明：此接口不保存历史；历史由前端调用 `/api/nav/chat` 管理。

## /api/ai/models · AI 模型列表

- 鉴权：用户登录
- 路由文件：`src/app/api/ai/models/route.ts`

### GET · 获取可用模型 ID

- 请求：无。
- 成功数据：`string[]`，仅包含 DeepSeek `/models` 返回项中的 `id`。
- 失败：未配置 API Key 时返回 `503`，上游异常时返回 `502`。
